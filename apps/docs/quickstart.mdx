In this guide, you'll build a simple AI application that generates text. We'll use OpenAI as an example, but you can use any provider (OpenAI, Anthropic, Qwen, etc.).

## Step 1: Install and Import

```bash npm
npm install @dysporium-sdk/openai
```

```bash pnpm
pnpm add @dysporium-sdk/openai
```

```typescript
import { createOpenAI } from '@dysporium-sdk/openai';
import { generateText } from '@dysporium-sdk/core';
```

## Step 2: Create a Provider

Create a provider instance:

**With OpenAI:**
```typescript
const openai = createOpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});
```

**With Anthropic:**
```typescript
import { createAnthropic } from '@dysporium-sdk/anthropic';

const anthropic = createAnthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});
```

**With Qwen:**
```typescript
import { createQwen } from '@dysporium-sdk/qwen';

const qwen = createQwen({
  apiKey: process.env.QWEN_API_KEY,
});
```

## Step 3: Generate Text

Use the `generateText` function to create completions:

```typescript
const result = await generateText({
  model: openai('gpt-5.1'),
  prompt: 'Write a haiku about TypeScript',
});

console.log(result.text);
```

## Complete Example

Here's a complete working example:

```typescript
import { createOpenAI } from '@dysporium-sdk/openai';
import { generateText } from '@dysporium-sdk/core';

async function main() {
  // Create provider
  const openai = createOpenAI({
    apiKey: process.env.OPENAI_API_KEY,
  });

  // Generate text
  const result = await generateText({
    model: openai('gpt-5.1'),
    prompt: 'Explain TypeScript in one sentence',
    maxTokens: 100,
    temperature: 0.7,
  });

  console.log('Response:', result.text);
  console.log('Usage:', result.usage);
}

main().catch(console.error);
```

## Using Messages

For conversational AI, use the messages API:

```typescript
const result = await generateText({
  model: openai('gpt-5.1'),
  messages: [
    { role: 'system', content: 'You are a helpful assistant.' },
    { role: 'user', content: 'What is TypeScript?' },
  ],
});
```

## Streaming Responses

Stream responses in real-time:

```typescript
import { streamText } from '@dysporium-sdk/core';

const result = await streamText({
  model: openai('gpt-5.1'),
  prompt: 'Tell me a story',
  onChunk: (chunk) => {
    if (chunk.type === 'text-delta') {
      process.stdout.write(chunk.textDelta);
    }
  },
});
```
