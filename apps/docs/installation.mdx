## Package Installation

Dysporium SDK is split into multiple packages for modularity:

### Option 1: All-in-One (Recommended)

Install a provider package which includes everything you need:

**For OpenAI:**
```bash npm
npm install @dysporium-sdk/openai
```

**For Anthropic:**
```bash npm
npm install @dysporium-sdk/anthropic
```

**For Qwen:**
```bash npm
npm install @dysporium-sdk/qwen
```

Each provider package includes:
- `@dysporium-sdk/core` - Core functionality
- Provider-specific implementation (OpenAI, Anthropic, or Qwen)
- `@dysporium-sdk/provider` - Provider interfaces

### Option 2: Individual Packages

Install packages separately for more control:

```bash npm
npm install @dysporium-sdk/core @dysporium-sdk/openai
```

## Environment Setup

### API Keys

Set your API keys as environment variables:

**For OpenAI:**
```bash
export OPENAI_API_KEY=your-api-key-here
```

**For Anthropic:**
```bash
export ANTHROPIC_API_KEY=your-api-key-here
```

**For Qwen:**
```bash
export QWEN_API_KEY=your-api-key-here
```

Or create a `.env` file:

```env
OPENAI_API_KEY=your-api-key-here
ANTHROPIC_API_KEY=your-api-key-here
QWEN_API_KEY=your-api-key-here
```

### TypeScript

The SDK is written in TypeScript and includes full type definitions. No additional types package needed.

### Node.js Version

Requires Node.js 18+ for native async/await and modern JavaScript features.

## Verify Installation

Test your installation:

**With OpenAI:**
```typescript
import { createOpenAI } from '@dysporium-sdk/openai';

const openai = createOpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

console.log('Dysporium SDK installed successfully!');
```

**With Anthropic:**
```typescript
import { createAnthropic } from '@dysporium-sdk/anthropic';

const anthropic = createAnthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

console.log('Dysporium SDK installed successfully!');
```

**With Qwen:**
```typescript
import { createQwen } from '@dysporium-sdk/qwen';

const qwen = createQwen({
  apiKey: process.env.QWEN_API_KEY,
});

console.log('Dysporium SDK installed successfully!');
```
