Providers are the bridge between Dysporium SDK and AI model APIs. They handle authentication, API calls, and model-specific configurations.

## What is a Provider?

A provider is an implementation of the SDK's provider interface. It creates language model instances that can be used with core functions like `generateText` and `streamText`.

## Using Providers

### OpenAI Provider

The OpenAI provider supports GPT models:

```typescript
import { createOpenAI } from '@dysporium-sdk/openai';

const openai = createOpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

// Create a model instance
const model = openai('gpt-5.1');
```

### Anthropic Provider

The Anthropic provider supports Claude models:

```typescript
import { createAnthropic } from '@dysporium-sdk/anthropic';

const anthropic = createAnthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

// Create a model instance
const model = anthropic('claude-sonnet-4-5');
```

### Creating Model Instances

Providers are functions that return model instances:

```typescript
// Single model
const gpt5 = openai('gpt-5.1');

// With options
const gpt5Mini = openai('gpt-5-mini', {
  temperature: 0.7,
});
```

## Provider Configuration

Each provider accepts configuration options:

```typescript
const openai = createOpenAI({
  apiKey: process.env.OPENAI_API_KEY,
  baseURL: 'https://api.openai.com/v1', // Optional: custom base URL
  defaultHeaders: {
    'Custom-Header': 'value',
  },
});
```

## Using Multiple Providers

You can use multiple providers in the same application:

```typescript
import { createOpenAI } from '@dysporium-sdk/openai';
import { createAnthropic } from '@dysporium-sdk/anthropic';

const openai = createOpenAI({ apiKey: process.env.OPENAI_API_KEY });
const anthropic = createAnthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

// Use different providers for different tasks
const gpt5 = openai('gpt-5.1');
const claude = anthropic('claude-sonnet-4-5');
```

## Provider Interface

Providers implement a standard interface, making them interchangeable:

```typescript
type Provider = (modelId: string, options?: ModelOptions) => LanguageModel;
```

This means you can switch providers without changing your application code:

```typescript
// Works with any provider
const result = await generateText({
  model: openai('gpt-5.1'), // or anthropic('claude-3'), etc.
  prompt: 'Hello',
});
```

## Embedding Providers

Providers can also create embedding models:

```typescript
import { createOpenAIEmbedding } from '@dysporium-sdk/openai';

const embeddingProvider = createOpenAIEmbedding({
  apiKey: process.env.OPENAI_API_KEY,
});

const embeddingModel = embeddingProvider('text-embedding-3-small');
```
