## Provider Functions

### createAnthropic

Create an Anthropic language model provider.

```typescript
function createAnthropic(config: AnthropicConfig): AnthropicProvider
```

**Parameters:**

- `config.apiKey` - Your Anthropic API key (required)
- `config.baseURL` - Custom base URL (optional, defaults to `https://api.anthropic.com/v1`)
- `config.retry` - Retry configuration (optional)

**Returns:** `AnthropicProvider` function

**Example:**

```typescript
const anthropic = createAnthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

const model = anthropic('claude-sonnet-4-5');
```

## Extended Thinking

Claude 3.5+ models support extended thinking for complex problem-solving:

```typescript
const result = await generateText({
  model: anthropic('claude-3-5-sonnet-20241022'),
  prompt: 'Solve this complex problem step by step',
  thinking: {
    type: 'enabled',
    budgetTokens: 10000,  // Token budget for thinking process
  },
});
```

The extended thinking feature allows Claude to work through problems internally before providing a response, leading to better reasoning on complex tasks.
