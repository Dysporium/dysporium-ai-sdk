## Provider Functions

### createQwen

Create a Qwen language model provider.

```typescript
function createQwen(config: QwenConfig): QwenProvider
```

**Parameters:**

- `config.apiKey` - Your Qwen API key (required)
- `config.baseURL` - Custom base URL (optional, defaults to `https://dashscope.aliyuncs.com/api/v1`)
- `config.retry` - Retry configuration (optional)

**Returns:** `QwenProvider` function

**Example:**

```typescript
const qwen = createQwen({
  apiKey: process.env.QWEN_API_KEY,
});

const model = qwen('qwen2.5-72b-instruct');
```

**Example:**

```typescript

  const result = await generateText({ /* ... */ });
} catch (error) {
  if (error instanceof RateLimitError) {
    console.log('Rate limit exceeded');
    console.log('Retry after:', error.retryAfterMs, 'ms');
  } else if (error instanceof APIError) {

}
```

## Features

- **Full Qwen Support** - Latest Qwen 3, Qwen 2.5, and Qwen 2.0 models
- **Top-K Sampling** - Control token selection with top-k sampling
- **Diversity Controls** - Frequency and presence penalties for better outputs
- **Streaming** - Real-time streaming responses
- **Tool Calling** - Function calling support with parallel execution
- **Automatic Retries** - Built-in retry logic with exponential backoff
- **Type Safety** - Full TypeScript support

## Advanced Parameters

Qwen supports advanced generation parameters:

```typescript
const result = await generateText({
  model: qwen('qwen2.5-72b-instruct'),
  prompt: 'Write a creative story',
  temperature: 0.8,
  topK: 50,              // Top-k sampling
  frequencyPenalty: 0.5, // Reduce repetition
  presencePenalty: 0.3,  // Encourage new topics
  seed: 42,             // Reproducible outputs
  parallelToolCalls: true, // Parallel tool execution
});
```
