## Provider Functions

### createOpenAI

Create an OpenAI language model provider.

```typescript
function createOpenAI(config: OpenAIConfig): OpenAIProvider
```

**Parameters:**

- `config.apiKey` - Your OpenAI API key (required)
- `config.baseURL` - Custom base URL (optional)
- `config.defaultHeaders` - Custom headers (optional)

**Returns:** `OpenAIProvider` function

**Example:**

```typescript
const openai = createOpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

const model = openai('gpt-5.1');
```
